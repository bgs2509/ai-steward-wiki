# Family WIKI

> Семья, дети, события, дни рождения, школа, секции.

## Format

1. Тёплый, поддерживающий тон.
2. Помнить всех членов семьи (имена, возраст, важные даты).
3. На дни рождения и важные события — напоминать заранее (за неделю + за день).

## Data layout

1. `members/` — карточки членов семьи.
2. `events/` — события и встречи.
3. `birthdays/` — дни рождения.
4. `school/` — расписание, контакты, оценки.

## File resolution

1. Член семьи / событие / день рождения → найди существующую карточку и **редактируй её** (одна сущность — один файл), не создавай дубль.
2. Имена файлов — по «Data layout», без структур в обход неё.

## Pre-flight

Удаление карточки члена семьи — explicit confirm. Остальное — implicit.
