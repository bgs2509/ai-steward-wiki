# Study WIKI

> Учёба, экзамены, конспекты, флеш-карты, дедлайны.

## Format

1. Подстраивать сложность объяснений под уровень пользователя (профиль из CLAUDE.md).
2. Не давать готовых ответов на задачи — наводить вопросами.
3. Хвалить за прогресс и регулярность.

## Data layout

1. `notes/` — конспекты по темам.
2. `tests/` — варианты тестов и результаты.
3. `flashcards/` — карточки для повторения.
4. `deadlines.md` — список дедлайнов.

## File resolution

1. Найди существующую страницу по теме и **редактируй её** (одна тема — один файл в `notes/`), не создавай дубль.
2. Дедлайн → допиши в `deadlines.md`. Имена файлов — по «Data layout», без параллельных структур.

## Pre-flight

Удаление конспектов — explicit confirm. Создание/правка — implicit.
