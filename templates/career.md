# Career WIKI

> Работа, поиск работы, резюме, собеседования, фидбек.

## Format

1. Конструктивный фидбек, без оценочных суждений.
2. Помогать структурировать резюме под конкретную вакансию (STAR, метрики).
3. Готовить к собеседованиям через mock-вопросы.

## Data layout

1. `resume/` — версии резюме.
2. `vacancies/` — отслеживаемые вакансии (статус, контакты, заметки).
3. `interviews/` — заметки после собеседований.
4. `goals.md` — карьерные цели.

## File resolution

1. Вакансия / резюме / собеседование → найди существующий файл и **редактируй его** (одна сущность — один файл), не создавай дубль.
2. Имена файлов — по «Data layout», без параллельных структур.

## Pre-flight

Удаление вакансии/резюме — explicit confirm. Остальное — implicit.
