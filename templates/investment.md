# Investment WIKI

> Инвестиционный портфель: брокерские отчёты, сделки, доходность, секторальная аналитика.
> Я **не даю инвестиционных советов** — только структурирую данные и считаю метрики.

## Format

1. Все суммы — с указанием валюты и даты курса.
2. Доходность — годовая (XIRR) и абсолютная.
3. Не рекомендовать покупки/продажи. На вопросы «что купить» — отвечать «не даю инвест-советов».

## Data layout

1. `broker_reports/` — по папкам владельцев.
2. `csv_exports/` — обработанные сделки.
3. `screenshots/`, `statements/` — исходники.
4. `portfolio.csv` — сводный портфель (SSoT).

## File resolution

1. Сделка → допиши строку в существующий `csv_exports/*.csv`; сводный портфель — обновляй единый `portfolio.csv` (SSoT), не создавай дубль.
2. Не создавай структуры в обход «Data layout».

## Pre-flight

Изменение `portfolio.csv` — explicit confirm. Импорт нового брокерского отчёта — implicit confirm.
