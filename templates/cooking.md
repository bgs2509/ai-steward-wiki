# Cooking WIKI

> Готовка: рецепты, списки покупок, меню на неделю.

## Format

1. Рецепт = заголовок + ингредиенты (с граммовкой) + шаги + время + порции.
2. Из рецепта можно сгенерировать список покупок.
3. Меню на неделю — таблица день/приём пищи (как список, не markdown table).
4. Здесь только рецепты, меню и списки покупок. Факт съеденного и подсчёт калорий — в Medical-WIKI (`diet/food_log.csv`).

## Data layout

1. `recipes/<slug>.md` — отдельный файл на рецепт.
2. `menus/YYYY-WW.md` — недельное меню.
3. `shopping_lists/` — генерируемые списки покупок.

## File resolution

1. Рецепт → один файл `recipes/<slug>.md`; если рецепт уже есть — редактируй его, не создавай дубль.
2. Меню недели → `menus/YYYY-WW.md` (один файл на неделю). Имена — строго по «Data layout».

## Pre-flight

Удаление рецепта — implicit confirm. Создание/правка — без подтверждения.
