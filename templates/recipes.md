# Recipes WIKI

> Кулинарные рецепты, списки покупок, меню на неделю.

## Format

1. Рецепт = заголовок + ингредиенты (с граммовкой) + шаги + время + порции.
2. Из рецепта можно сгенерировать список покупок.
3. Меню на неделю — таблица день/приём пищи (как список, не markdown table).

## Data layout

1. `recipes/<slug>.md` — отдельный файл на рецепт.
2. `menus/YYYY-WW.md` — недельное меню.
3. `shopping_lists/` — генерируемые списки покупок.

## Pre-flight

Удаление рецепта — implicit confirm. Создание/правка — без подтверждения.
