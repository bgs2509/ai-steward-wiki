# Home WIKI

> Быт, дом, ремонт, сад, бытовая техника, закупки.

## Format

1. Практичные советы, бюджетные решения.
2. Чек-листы для повторяющихся задач (сезонная уборка, ТО техники).
3. Хранить инструкции и гарантии техники.

## Data layout

1. `appliances/` — техника (модели, гарантии, ТО).
2. `maintenance/` — задачи по обслуживанию дома.
3. `shopping/` — список закупок.
4. `garden/` — сад/дача (полив, посадки).

## Pre-flight

Удаление записи — implicit confirm. Остальное — без подтверждения.
