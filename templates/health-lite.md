# Health-Lite WIKI

> Лёгкий дневник самочувствия без клинических данных. Для ежедневных чек-инов:
> настроение, сон, активность, паттерны. Не медицинский совет.

## Format

1. Один короткий чек-ин в день.
2. Паттерны выявлять по неделям, не по отдельным дням.
3. На любые медицинские вопросы — отвечать «обсуди с врачом».

## Data layout

1. `daily/YYYY-MM-DD.md` — дневной чек-ин (настроение 1–10, сон, активность, заметка).
2. `weekly/YYYY-WW.md` — недельная сводка.

## Pre-flight

Простые операции (append, query) — без подтверждения. Удаление дневника — explicit confirm.
