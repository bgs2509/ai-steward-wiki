# Predictive Replies

**Тип:** concept
**Статус:** draft
**Источники:** диалог 2026-05-08, [time-tracker](../entities/time-tracker.md)

## Суть

К каждому вопросу «что делал?» прикрепляются **3 инлайн-кнопки** с наиболее вероятными ответами + кнопка «другое». Юзер тапает одну — ответ зафиксирован за 1 жест, без набора текста.

## Правила

1. **Всегда ровно 3 кнопки-предсказания** + 1 «другое». Меньше нельзя — нарушит UX-договорённость.
2. **Источник предсказаний** — история ответов на тот же слот (день недели + час) за последние N недель.
3. **Сортировка** — по убыванию частоты в этом слоте.
4. **Fallback при пустой истории** — дефолтный набор по времени дня:
   1. Утро (06–11): «работа», «завтрак», «зарядка»
   2. День (11–17): «работа», «обед», «дорога»
   3. Вечер (17–23): «дома», «спорт», «ужин»
5. **«Другое»** — свободный текст. Ответ попадает в историю и через N повторов поднимается в топ-3.

## Где применяется

1. Периодический опрос «что делал?» из [time-tracker](../entities/time-tracker.md).
2. Follow-up «сделал?» из [mandatory-checkins](../concepts/mandatory-checkins.md) — там кнопок только 2 (да/нет), это другой паттерн, не путать.
