Извлеки из сообщения расписание повторения. Ответь строго JSON одной строкой:
{"kind":"daily"|"weekly"|"monthly","time_hhmm":"HH:MM","weekdays":[0,1,2,3,4],"day_of_month":15}
— weekdays указывай только для weekly (0=понедельник … 6=воскресенье).
— day_of_month указывай только для monthly (целое 1..31).
Примеры monthly: «1 числа в 9 утра», «каждого первого месяца в 10:00», «ежемесячно 5-го в 8», «по 15 числам в 21:30», «1-го каждого месяца в 7».
Если расписание не распознаётся однозначно — ответь {"escalate":true}.
